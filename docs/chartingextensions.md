# ChartingExtensions


The **ChartingExtensions** class provides helper methods useful for converting a pixel coordinate from application-specific pixels (i.e., WPF coordinates) to Device Independent Pixels.
![note image](/_next/image?url=%2F_next%2Fstatic%2Fmedia%2FNote.57c6078c.svg&w=64&q=75)

## Note


More information about the differences between application pixels and device pixels can be found on the [Working with Pixel Coordinates](https://developer.ninjatrader.com/docs/desktop/working_with_pixel_coordinates) page.


## [ChartingExtensions Helper Methods](https://developer.ninjatrader.com/docs/desktop/chartingextensions#chartingextensions-helper-methods)


| Method | Description |
| --- | --- |
| [ConvertFromHorizontalPixels](https://developer.ninjatrader.com/docs/desktop/convertfromhorizontalpixels) | Converts a horizontal coordinate (x) from device pixels to application pixels |
| [ConvertFromVerticalPixels](https://developer.ninjatrader.com/docs/desktop/convertfromverticalpixels) | Converts a vertical coordinate (y) from device pixels to application pixels |
| [ConvertToHorizontalPixels](https://developer.ninjatrader.com/docs/desktop/converttohorizontalpixels) | Converts a horizontal coordinate (x) in application pixels to device pixels |
| [ConvertToVerticalPixels](https://developer.ninjatrader.com/docs/desktop/converttoverticalpixels) | Converts a vertical coordinate (y) in application pixels to device pixels |
