# User-Based Vendor Licensing

Source: https://developer.ninjatrader.com/docs/desktop/user_based_licensing

---

The following section describes how commercial vendors can license proprietary indicators and strategies to customers through user-based licensing.


In the past, vendors were provided licensing through the NinjaTrader Ecosystem that required customers to verify their product license via their machine ID (a generated string of characters that identifies a physical computer). However, machine IDs take more time to verify, can change because of updates, and don't account for Virtual Machine (VM) instances.


Now, with user-based licensing, vendors can assign licenses and free trials to a user's NinjaTrader account. A user's product license from a vendor is now tied to their login for NinjaTrader and no longer tied to a single installation of the NinjaTrader desktop application. This provides easier management of licensing for vendors, as well as greater ease of use and adoption by users of vendor products.


If you are new to commercial licensing, please see [Commercial Distribution](https://developer.ninjatrader.com/docs/desktop/commercial_distribution).
