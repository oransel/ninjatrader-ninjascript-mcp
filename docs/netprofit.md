# NetProfit


## [Definition](https://developer.ninjatrader.com/docs/desktop/netprofit#definition)


Returns the net profit.


## [Property Value](https://developer.ninjatrader.com/docs/desktop/netprofit#property-value)


A **double** value that represents the net profit.


## [Syntax](https://developer.ninjatrader.com/docs/desktop/netprofit#syntax)


`<tradecollection>.TradesPerformance.NetProfit`


## [Examples](https://developer.ninjatrader.com/docs/desktop/netprofit#examples)


```csharp
protected override void OnBarUpdate()
// Print out the net profit of all trades
Print("Net profit is: " + SystemPerformance.AllTrades.TradesPerformance.NetProfit);
```
