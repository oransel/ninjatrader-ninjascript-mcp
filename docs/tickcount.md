## [Definition](https://developer.ninjatrader.com/docs/desktop/tickcount#definition)


Returns the total number of ticks of the current bar processing.
![note image](/_next/image?url=%2F_next%2Fstatic%2Fmedia%2FNote.57c6078c.svg&w=64&q=75)

## Note


For historical usage, you must use **Calculate.OnEachTick** with [**TickReplay**](https://developer.ninjatrader.com/docs/desktop/developing_for_tick_replay) enabled; otherwise a value of 1 will be returned.


## [Property Value](https://developer.ninjatrader.com/docs/desktop/tickcount#property-value)


A long value that represents the total number of ticks of the current bar.


## [Syntax](https://developer.ninjatrader.com/docs/desktop/tickcount#syntax)


`Bars.TickCount`


## [Examples](https://developer.ninjatrader.com/docs/desktop/tickcount#examples)


```csharp
// Prints the tick count to the output window
Print("The tick count of the current bar is " + **Bars.TickCount**.ToString());
```
