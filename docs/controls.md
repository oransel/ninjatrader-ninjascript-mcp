# NinjaTrader Controls


## [NinjaTrader Controls](https://developer.ninjatrader.com/docs/desktop/controls#ninjatrader-controls)


The following section contains controls that are native **NinjaTrader** controls. To fully integrate your Add On within **NinjaTrader** it is recommended to use these controls as opposed to building your own when possible.
![note image](/_next/image?url=%2F_next%2Fstatic%2Fmedia%2FNote.57c6078c.svg&w=64&q=75)

## Note


For cleaning up these resources, please see the **NTTabPage.Cleanup()** method.


| Control | Description |
| --- | --- |
| [AccountSelector](https://developer.ninjatrader.com/docs/desktop/accountselector) | AccountSelector can be used as a UI element users can interact with for selecting accounts. |
| [AtmStrategySelector](https://developer.ninjatrader.com/docs/desktop/atmstrategyselector) | AtmStrategySelector is a UI element users can interact with for selecting ATM Strategies. |
| [InstrumentSelector](https://developer.ninjatrader.com/docs/desktop/instrumentselector) | InstrumentSelector is a UI element users can interact with for selecting instruments. This can be used with instrument linking between windows. |
| [IntervalSelector](https://developer.ninjatrader.com/docs/desktop/intervalselector) | IntervalSelector is a UI element users can interact with for selecting intervals. This can be used with interval linking between windows. |
| [TifSelector](https://developer.ninjatrader.com/docs/desktop/tifselector) | TifSelector can be used as a UI element users can interact with for selecting TIF. |
| [QuantityUpDown](https://developer.ninjatrader.com/docs/desktop/quantityupdown) | QuantityUpDown can be used as a UI element users can interact with for selecting quantity. |
